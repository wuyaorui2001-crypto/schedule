# SYSTEM - 日程管理 AI操作指南

> 本文件是AI接手日程项目的必读指南
> 版本: v2.0 | 更新: 2026-03-18

## 1. 项目一句话定义

**个人日程与任务管理系统，支持自然语言输入、待办追踪、提醒通知，采用Markdown结构化存储。**

---

## 2. 核心功能（参考 todo-manager + reminder）

### 自然语言输入（like reminder）
- "后天上午10点开会" → 自动解析日期时间
- "下周三下午3点交报告" → 自动计算日期
- "每月1号还信用卡" → 支持重复任务

### 待办管理（like todo-manager）
- 添加待办
- 标记完成 [x]
- 删除待办
- 查看列表

### 提醒通知（like reminder，改用飞书）
- 默认提醒：24h/1h/10m 前
- **使用飞书消息通知**（已配置）
- 可查询"最近有什么安排"

## 3. AI接手后的标准SOP流程

### 任务分类（重要）

| 类型 | 特征 | 时间格式 | 示例 |
|-----|------|---------|------|
| **⏰ 定时任务** | 需要提醒，有具体时间 | YYYY-MM-DD HH:MM | 18:00 吃饭 |
| **📝 普通待办** | 当天完成即可，无具体时间 | YYYY-MM-DD | 今天看电影 |

### 场景A：自然语言添加日程
1. **解析输入**：识别时间、事件、是否需要提醒
   - 有具体时间（几点几分）→ **定时任务**，用完整时间格式
   - 无具体时间（今天/明天）→ **普通待办**，只用日期
2. **确认解析**：向用户确认理解正确
3. **存储记录**：按年月分文件追加
4. **设置提醒**：创建 cron 任务（仅定时任务）

### 场景B：查看待办（分类展示）
1. 读取当前月份文件
2. 分类展示：
   ```
   [⏰ 定时任务]
   1. 18:00 吃饭 #生活
   2. 19:00 记账 #财务
   
   [📝 普通待办]
   1. 看电影《云图》 #娱乐
   2. 写日记 #思考
   3. 看《风险投资史》两章 #阅读
   ```
3. 按优先级排序（P0 → P3）

### 场景C：完成待办
1. 找到对应记录
2. 状态改为 [已完成]
3. 添加完成时间（如需要）
4. 更新统计

### 场景E：设置飞书提醒（定时任务）
1. 读取定时任务时间
2. 使用 `cron` 创建定时任务
3. 提醒内容通过 **飞书消息** 发送
4. **提醒格式规则**（必须遵守）：
   ```
   ⏰ 提醒
   {事项}！
   —《schedule》
   ```
   示例：
   ```
   ⏰ 提醒
   吃饭！
   —《schedule》
   
   ⏰ 提醒
   记账！
   —《schedule》
   ```
5. 禁止添加其他多余内容

## 4. 数据格式（优化版）

### 日程记录格式
```markdown
- 2026-03-18 18:00 | [待办] | [P1高] | 吃饭 | #生活 | 提醒:准时
- 2026-03-18 | [待办] | [P2中] | 看电影《云图》 | #娱乐
- 2026-03-19 10:00 | [已完成] | [P0紧急] | 报告 | #工作 | 完成:2026-03-19 09:30
```

### 字段说明
| 字段 | 格式 | 示例 | 说明 |
|-----|------|------|------|
| 时间 | YYYY-MM-DD 或 YYYY-MM-DD HH:MM | 2026-03-18 或 2026-03-18 14:00 | **提醒事项**用完整时间，**普通待办**只用日期 |
| 状态 | [待办/进行中/已完成/取消] | [待办] | 当前状态 |
| 优先级 | [P0紧急/P1高/P2中/P3低] | [P1高] | 优先级标记 |
| 标题 | 事件描述 | 会议 | 简短描述 |
| 标签 | #分类 | #工作 | 分类标签 |
| 提醒 | 提醒:时间 | 提醒:15分钟前 | 仅提醒事项需要 |
| 完成时间 | 完成:时间 | 完成:2026-03-19 09:30 | 已完成时记录 |

## 5. 自然语言解析规则

### 时间识别
| 输入 | 解析结果 |
|-----|---------|
| 今天 | 2026-03-18 |
| 明天 | 2026-03-19 |
| 后天 | 2026-03-20 |
| 下周三 | 下周的周三 |
| 下个月2号 | 下个月2号 |
| 上午10点 | 10:00 |
| 下午3点 | 15:00 |
| 晚上8点 | 20:00 |

### 重复识别
| 输入 | 解析结果 |
|-----|---------|
| 每天 | 重复:每天 |
| 每周五 | 重复:每周五 |
| 每月1号 | 重复:每月1号 |
| 每年 | 重复:每年 |
